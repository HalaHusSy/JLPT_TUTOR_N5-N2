# JLPT Tutor — N5 ถึง N1

Web app สำหรับติวสอบวัดระดับภาษาญี่ปุ่น JLPT ทุกระดับ N5 → N1 ออกแบบให้เปิดใช้งานในเบราว์เซอร์ได้เลย ไม่ต้องตั้ง server

## ฟีเจอร์

- 📖 **คำศัพท์** — รายการคำพร้อม flashcard เปิด-ซ่อนเฉลย (Thai meaning + romaji + 3 ประโยคตัวอย่าง)
- 漢 **คันจิ** — แสดงตัวอักษรขนาดใหญ่ พร้อม on'yomi / kun'yomi และตัวอย่างการใช้
- 📝 **ไวยากรณ์** — รวม pattern ตามระดับ N5–N1 พร้อมตัวอย่างประโยค
- 📚 **บทอ่าน** — บทความสั้นพร้อมคำถามท้ายบทและเฉลยภาษาไทย
- 🧪 **Mock Exam** — ข้อสอบจำลองแบบ 4 ตัวเลือก พร้อมเฉลย คำอ่าน และคำอธิบายไทย
- 🎯 **Quiz mode** — สุ่ม 10 ข้อจากชุดข้อมูลที่เลือก ตรวจคะแนนทันที
- 📊 **สถิติ** — ติดตามจำนวนคำที่เปิดเฉลยแล้วในแต่ละระดับ บันทึกใน localStorage

## วิธีเปิดใช้

```
ดับเบิลคลิกไฟล์ index.html ในเบราว์เซอร์
```

ไม่ต้องตั้ง server — ทุกอย่างเป็น static HTML/CSS/JS โหลดจากไฟล์ในเครื่อง

## โครงสร้างไฟล์

```
JLPT_TUTOR_N5-N2/
├── index.html              # หน้าหลัก
├── css/style.css           # ธีม dark สีแดง-น้ำเงิน
├── js/app.js               # ตัวจัดการ UI, quiz, mock exam, สถิติ
└── data/
    ├── vocab-n5.js         # คำศัพท์ N5 (ชุดเริ่มต้น)
    ├── vocab-n5-pdf-*.js   # ขยายจาก TRY! N5 PDF (~330 คำ แบ่งตามแถว ก-อ)
    ├── vocab-n4.js ... n1.js
    ├── kanji-n5.js         # คันจิ N5 ชุดเริ่มต้น
    ├── kanji-n5-pdf.js     # คันจิ N5 ขยายจาก PDF
    ├── kanji-n4.js ... n1.js
    ├── grammar-n5.js       # ไวยากรณ์ N5 ชุดแรก
    ├── grammar-n5-pdf.js   # ไวยากรณ์ N5 ขยายเพิ่ม
    ├── grammar-n4.js ... n1.js
    ├── reading.js          # บทอ่านชุดเริ่มต้น
    ├── reading-n5-pdf.js   # บทอ่าน N5 เพิ่ม
    └── mock.js             # Mock Exam ทุกระดับ
```

## การเพิ่ม / แก้คำศัพท์

แต่ละไฟล์ใน `data/` คือ JavaScript ที่ push เข้า array global เช่น `window.vocabN5`:

```js
window.vocabN5 = window.vocabN5 || [];
window.vocabN5.push(
  { word: "新しい", reading: "あたらしい", romaji: "atarashii", meaning: "ใหม่",
    examples: [
      { jp: "新しい本。", romaji: "Atarashii hon.", th: "หนังสือใหม่" },
      ...
    ]
  },
);
```

ระบบมี dedup ใน `js/app.js` คำซ้ำจะถูกกรองอัตโนมัติ

## หมายเหตุ

- ข้อสอบ JLPT จริงเก่าเป็นลิขสิทธิ์ของ JEES (เปิดเผยแค่ตัวอย่าง) Mock Exam ในแอปนี้ออกแบบขึ้นใหม่ตามรูปแบบ JLPT
- คำศัพท์ N5 ขยายจาก TRY! 日本語能力試験 N5 ฉบับภาษาไทย (สมาคมส่งเสริมเทคโนโลยีไทย-ญี่ปุ่น)
